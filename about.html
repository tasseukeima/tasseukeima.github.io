About Section
Updated on 5/14/2021 at 6:40 PM
There are four main directories on this website.

Index:
This is the first page one is directed to when they arrive on this site.


Home:
This page is meant to display information regarding the latest updates to tasseukeima.github.io.
Information about the overall functionality can be found here too.


Main Board:
This page is meant to replicate a conventional blog. I will often update this area with my personal thoughts about my life and other things.
I may also talk about new features that could be implemented in the future.


Archived:
There are 3 main sections to this page.

The "Internal Branch" is locked, but this is a page that I use to test new features on. The main reason it is locked is because half of the time it it broken and not functional.

The "Experimental Branch" is open for access, and used to give a snapshot of features that made it out of Internal.

The "Public Branch" is open and used to archive the changes made to the site, as well as give information about the current development status.
